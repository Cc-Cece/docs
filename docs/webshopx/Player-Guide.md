# 玩家使用指南

[返回首页](./Home)

## 1. 首次使用

### 1.1 绑定与设置密码

在游戏内执行：

```text
/ws password <新密码>
```

成功后可使用 Minecraft 用户名 + 该密码登录网页。

### 1.2 登录网页

- 玩家端：`/`（如 `http://host:8819/`）
- 输入用户名和密码。

## 2. 钱包与兑换

### 2.1 货币类型

- `SHOP_COIN` （具体名称可能有所不同）
- `GAME_COIN` （具体名称可能有所不同）

`GAME_COIN` 可能由 Vault 经济插件驱动，具体由服主配置决定。

### 2.2 钱包功能

- 查看余额
- 查看最近流水
- 执行双币兑换（仅当对应方向启用）
- 使用兑换码入账

## 4. 玩家市场

### 6.1 买家侧

- 可筛选币种、价格区间、材质、关键字。
- 一口价模式可直接购买。
- 拍卖模式需出价；资金会冻结，未中标自动退回。

### 6.2 卖家侧

- 可通过 GUI 上架、暂停、恢复、下架。
- 可查看最近店铺成交记录：

```text
/ws market logs [count]
```

- 供货箱上架支持自动补货与手动刷新（由市场设置控制）。

## 4. 退款规则

一般情况下满足下面任一条件可退款：

- 服务器是否启用 `refund-undelivered-enabled`且玩家未发放、未领取或未核销
- 处在冷静期内（冷静期内商品不会发放）

若可退款，提交后余额会返还到原支付货币钱包。

## 7. 常见问题

### 7.1 我买了东西但没到账

检查订单状态

执行：

```text
/ws claim all
```

若仍失败，请检查：

- 背包是否已满
- 服务器是否短时卡顿

### 7.2 竞拍后余额变少

竞拍会冻结出价资金。未中标或拍卖取消时会自动退回。

### 7.3 兑换失败

常见原因：

- 兑换方向未开启
- 输入数量无效
- 钱包余额不足

# 上架限制

> 这页只回答一个问题：为什么你发现有时你无法正常上架。

## 1. 先理解一件事

每次你创建或修改市场挂单时，系统都会做一次“上架检查”。
检查不通过，就会直接拒绝，并提示错误码。

## 2. 最常见的限制类型

1. 物品被禁卖。
2. 交易方式不允许（例如只允许卖单，不允许买单）。
3. 币种不允许。
4. 标签不允许或标签无效。
5. 你缺少对应权限。

## 3. 你可以怎么快速排查

按这个顺序查，最快：

1. 先确认你在做的是“买单”还是“卖单”，再确认该模式是否开放。
2. 换一个常用币种再试一次。
3. 去掉自定义标签再试一次。
4. 换一个普通物品测试（例如圆石、木头），排除物品被禁卖。
5. 如果你有管理权限，确认自己不是被权限规则拦截。

## 4. 常见错误码对照

| 错误码 | 意思 |
| --- | --- |
| `limitation_item_forbidden` | 这个物品不允许上架 |
| `limitation_side_not_allowed` | 当前挂单方向不允许 |
| `limitation_trade_mode_not_allowed` | 当前交易模式不允许 |
| `limitation_currency_not_allowed` | 当前币种不允许 |
| `invalid_tag` | 标签不存在或不被允许 |

## 5. 关于“绕过限制”权限

权限 `webshop.market.limitation.bypass` 可以跳过限制检查。

:::warning[注意]
这个权限等于“跳过市场风控”，只建议给服主管理账号，不建议给普通玩家。
:::

## 6. 给普通玩家的建议

1. 遇到报错时把“错误码 + 你在做什么”发给管理员，处理最快。
2. 如果你今天能上架、明天不能，通常是服务器规则更新了，不是你操作错了。

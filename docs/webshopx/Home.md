# WebShopX Wiki

[中文文档入口](./Overview)

WebShopX 是一个面向 Paper / Spigot 服务器的 Web 商店插件，整合了官方商城、玩家市场、钱包、订单流转、发货和后台管理。  
WebShopX is a web-first commerce plugin for Paper / Spigot servers, combining official shop, player market, wallet, order flow, delivery, and admin console.
